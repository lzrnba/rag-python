## 技术选型

| 组件 | 选型 | 原因 |
|:--|:--|:--|
| LLM | Qwen2.5-7B（Ollama） | 本地部署，中文效果好 |
| 框架 | LangGraph 0.0.20 | 支持状态图和闭环迭代 |
| API | FastAPI | 高性能，自动生成文档 |
| 向量库 | FAISS | 轻量，无需额外服务 |
| 关键词检索 | BM25（rank-bm25） | 精确匹配补充语义检索 |
| Embedding | bge-m3（Ollama） | 多语言，中文效果优秀 |
| 前端 | 原生 HTML/CSS/JS | 无依赖，直接打开 |

## 系统目标

- 响应时间 < 3s
- 支持 PDF / Word / MD / TXT 文档
- 支持热重载文档（无需重启服务）
- 闭环迭代检索，最多 3 次
- 可解释的推理过程（CoT）
